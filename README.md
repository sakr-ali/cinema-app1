تطبيق السينما باستخدام Flutter



\##  الوصف



هذا التطبيق هو مشروع مبني باستخدام Flutter يوضح أساسيات التنقل بين الشاشات (Navigation) وإدارة البيانات. يتيح للمستخدم تصفح قائمة من الأفلام، عرض تفاصيل كل فيلم، وإضافته إلى قائمة المشاهدة (Watchlist) بسهولة.



\##  المميزات



\*  التنقل بين الشاشات باستخدام Navigator

\*  إرسال واستقبال البيانات بين الصفحات

\*  عرض قائمة أفلام مع صور ووصف

\*  صفحة تفاصيل لكل فيلم

\*  إمكانية إضافة الأفلام إلى قائمة المشاهدة

\*  واجهة مستخدم داكنة وجذابة
